Grade Archiver
-------------------
This plugin archives grades when courses are completed so that they will be saved when the user unenrolls from a course after completion.
DB structure
-------------
grades (keeps track of final course grades)
user id
course id
pretest score
posttest score
datetime


Events
-------
The events listened for are:

    Course completion - add to grades table
 
UI
---
We are going to (tentatively) use https://docs.moodle.org/28/en/Configurable_reports for the UI.
